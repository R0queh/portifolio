Conhecimentos

HTML

Uma linguagem de marcação utilizada na construção de páginas na Web, (usada nesse site que está vendo no momento) sei os conceitos das tags e estudo para aprender como aplica-las em sites e aplicações WEB de modo autonomo.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Aprendi os conceitos de algoritmo tenho conhhecimento sobre Operadores lógicos, operadores relacionais, laço condicional, laço de repetição, array

Bootstrap

Framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

CSS

uma ferramente de estilização que é um mecanismo para adicionar estilo a um documento web, usei para dar o visual do site que esta vendo no momento e que venho aprendendo e melhorarando em conceitos de estilização.s

Java

Linguagem que uso para programar a parte do backend usando o framework Spring para criar APIs Rest. Além disso aprendi sobre as data anotations, POO, Operadores lógicos, operadores relacionais, laço condicional, laço de repetição, matriz e vetor.

Git e GitHub

Ferramenta de verssionamento de código que uso para fazer os projetos, sabendo utilizar os commitss, fork, git add, pull, merge e o conceito de branch, a ponto de conseguir participar de projetos colaborativos
